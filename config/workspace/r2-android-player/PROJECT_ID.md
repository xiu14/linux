# Project Identity: R2 Android Player

This directory is only for the R2/AList Android video player.

- App name: R2 Player
- Android applicationId: `com.xiuzai.r2player`
- Kotlin package root: `com.xiuzai.r2player`
- Primary purpose: browse AList/R2 media entries and play videos with ExoPlayer.
- Native features here include AList API access, thumbnails, ExoPlayer playback, and player gestures.

Do not add SillyTavern WebView shell, ST bridge, Fluid Cloud, promoted notifications, or Tavern settings code to this project.
Those belong in `/config/workspace/st-android-shell`.
